Core Features

  • Multi-provider support (Kalshi & Polymarket)
  • Standardized API with response envelopes
  • Mock fallback for development
  • Async HTTP with retry logic
  • Type-safe Pydantic models
  • Comprehensive error handling
  • Interactive OpenAPI documentation
  • Production-tier architecture

Technology Stack

Python 3.11 FastAPI Pydantic httpx uvicorn pytest black flake8 isort

API Endpoints

GET/api/v1/health
GET/api/v1/markets
GET/api/v1/markets/{id}
GET/api/v1/markets/{id}/price
GET/api/v1/markets/{id}/timeseries
GET/api/v1/markets/{id}/orderbook
GET/api/v1/markets/{id}/events
POST/api/v1/ingest/{provider}/sync

Quality Assurance

  • Automated CI pipeline (GitHub Actions)

  • Unit tests with pytest
  • Mock testing without external APIs

Quick Start

Run locally in under 2 minutes:

git clone https://github.com/asapabhii/omniverse-market-api.git
cd omniverse-market-api
pip install -r requirements.txt
uvicorn api.main:app --reload
